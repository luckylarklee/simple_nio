# simple_nio

simple_nio (https://github.com/luckylarklee/simple_nio)
================================================

#1.简介
	#该库是基于Java NIO，提供简单高效异步(asynchronous)I/O框架，
	#高效的TCP/UDP/Pipe通信
	#提供流量控制(Flow control)
	#支持过滤机制，例如SSLFilter / StatisticFilter / LogFilter
	#活动对象模式 支持轮询，阻止和回调
	#支持可插拔内存管理和调度策略
	#支持优先发送
	#支持JMX
	#易于配置和测试.

#2.准备运行环境
	#- JDK 5+
